Seabass for SailfishOS v0.7.0

Release notes

The release contains two major changes:

  • Sailfish WebView is used to render editor instead of QtWebKit
  • Codemirror replaces Ace as text editor component

The changes should result in much better application performance overall, but have a few side effects:

  • while syntax highlight is still supported for over 100 languages, the list of supported languages has changed (see Codemirror docs for details)
  • pulley menus don't work with on-screen keyboard opened
  • autocompletion options and code snippets have changed
  • better support for text selection
  • there might be issues with editor rendering and cursor positioning in SFOS 4.3, but I expect significant improvements with future WebView updates

Please let me know if you have any issues: Seabass on Github.
