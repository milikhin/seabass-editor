Seabass for UBports v0.2.0

Release notes

This release features code autocompletion and snippets (for some languages) and a new navigation toolbar to make it easier to use the new features with on-screen keyboard!

Your feedback on github or in the dedicated topic on the UBports forum is very appreciated!

Funding

Seabass is a free software. If you'd like see the project growing, you might want to support the development.

The current goal is to have $65 monthly to keep the project development speed and fund a full workday once a month solely for the awesome new features! It doesn't really matter how much you pay, the support is what matters:

  • Liberapay (pay what you want)
  • Patreon (fixed $2 tier)
