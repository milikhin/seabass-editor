Seabass for UBports v0.1.1

Release notes

The current release provides ability to edit existing files. This is an early preview version, so things might be broken and some core functionality such as creating new files is yet to be implemented.

Features

  • Syntax highlighting for over 120 programming/markup languages
  • Multiple tabs
  • Undo/Redo
  • Light and Dark themes
