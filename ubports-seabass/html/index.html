Seabass2 v2.2.0

Features:

  • Syntax highlighting for over 100 programming/markup languages
  • Context menu to create/rename/delete files
  • Read indentation preferences from .editorconfig files
  • Open files with the Seabass using the File Manager application
  • Create and build projects with Clickable (requires Libertine)
  • Experimental intellisense support for C++, JavaScript, Python and TypeScript (requires Libertine)

New:

  • Added Find/Replace functionality

Known issues:

  • Installation of built packages doesn't work
  • Creating the build container might take several tries
