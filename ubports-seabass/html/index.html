Seabass2 v0.5.0

Release notes

v0.5 adds ability to build existing QML, C++, Python and HTML projects with Clickable. To build a project open clickable.json and click on the "Build" icon.

Please note that this feature is very-very-very experimental. There are a few requirements:

  • all project files should be located inside ~/Downloads or ~/Documents
  • your device should have python3-libertine and libertine-tools installed (should be preinstalled with the operating system)

Your feedback on github or in the dedicated topic on the UBports forum is very appreciated!
