Seabass2 v0.7.0

Release notes

v0.7 adds find/replace functionality.

v0.6 introduced support for setting indentation preferences using .editorconfig files. indent_style and indent_size options are supported.

Build tools

This release features (very-very-very experimental) ability to build existing QML, C++, Python and HTML projects with Clickable. To build a project open clickable.json file and click on the "Build" icon.

Please see the support topic for details. Your feedback is very appreciated!
